# meiryo7743.net-normalize-css

[Meiryo7743.net](https://meiryo7743.net) 用に作ったクラスレスなリセット CSS です。User agent stylesheet を上書きすることで，各ブラウザーにおける要素の初期スタイルを統一します。また，特定の言語に対し適切なスタイルを与えます。
